# Artifacts (local-only)

Use esta pasta para guardar artefatos temporários, dumps de debug e execuções ad-hoc
que não devem ser versionadas (outputs grandes, checkpoints descartáveis, logs, etc.).
Tudo dentro de `artifacts/` está listado no `.gitignore`, então fique à vontade para
criar subpastas conforme a tarefa — o conteúdo será limpo/local por padrão.
